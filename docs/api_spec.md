# API Definitions (Internal & Gateway)

Base URL: `/api/v1`

## Bot Management
- `GET /bots`: List all active/inactive bots.
- `POST /bots`: Create a new bot instance.
- `PATCH /bots/{id}/status`: Change bot status (start/stop/pause).
- `GET /bots/{id}/logs`: Get detailed execution logs for a specific bot.

## Market Data
- `GET /market/prices`: Get real-time prices from Redis cache.
- `GET /market/history`: Fetch OHLCV data from TimescaleDB.

## Trading & Strategy
- `GET /strategies`: List available strategy templates.
- `POST /strategies/backtest`: Trigger a backtesting job.
- `GET /strategies/backtest/{id}/results`: Retrieve backtest metrics and charts.

## AI Insights
- `POST /ai/explain-trade/{trade_id}`: Request an AI explanation for a specific trade.
- `GET /ai/market-report`: Get a summary of market conditions generated by local LLM.

## Risk
- `POST /risk/kill-switch`: Emergency stop for all bots.
- `GET /risk/exposure`: Current exposure across all assets and accounts.

## Security
- All requests require a valid JWT.
- API keys are handled only via POST (encrypted) and never returned in full.
